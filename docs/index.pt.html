❓ Perguntas frequentes

Isso é realmente grátis?

Sim. Os nós são operados por voluntários de terceiros que publicam suas próprias assinaturas gratuitas. Nós não operamos nenhum servidor — apenas testamos, classificamos e reempacotamos o que já é público.

Quão atualizados são os dados?

Uma GitHub Action roda a cada hora: puxa todas as fontes, faz sondagem TCP+TLS em cada nó, descarta os mortos, ordena por latência e comita os novos arquivos.

Posso confiar nesses nós?

Nós gratuitos veem todo o seu tráfego. Nunca os use para banco, login ou algo sensível. Bom para driblar bloqueios geográficos em conteúdo público. Use seu próprio VPS ou serviço pago para privacidade real.

Por que alguns nós listados falham?

Verificamos acessibilidade TCP e handshake TLS, mas um nó ainda pode ter cota esgotada, roteamento errado ou certificado expirado. Tente alguns; o grupo selector oferece alternativas.
