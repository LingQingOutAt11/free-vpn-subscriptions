🪜 操作步骤

  1. 安装 Shadowrocket

    Shadowrocket (小火箭) 是 App Store 付费应用,约 $2.99。中国区 App Store 没有,得用非中国区 Apple ID 登录(美区、日区、港区都行)。没非中国区账号的,改用免费 Loon 或 sing-box (免费)。

  2. 复制 v2ray 订阅链接

    在首页,长按 v2rayN / v2rayNG / Shadowrocket 那行 URL,选 复制。这是 v2ray base64 格式,Shadowrocket 认得。

  3. 通过 Subscribe 导入

    打开 Shadowrocket → 点右上 + → Type → Subscribe。把 URL 粘进 URL,名字随便,点 Save。

  4. 更新订阅

    回主界面,下拉刷新,或点订阅那一行 → Update。能看到几十个节点。

  5. 选连接模式

    底部点 Config。Global Proxy 是全局;Proxy (Rule-based) 是按规则走(推荐)。回主屏幕拨开关连接。第一次 iOS 会弹 VPN 权限,点允许。

  6. 验证

    Safari 打开 ipinfo.io,国家应该和所选节点一致。如果没变,换一个节点——iOS 的 TLS 栈比较严,部分免费节点在 iOS 上会抽风。

💡 常见问题排错

App Store 显示 Shadowrocket 在该地区不可用

需要美区/日区/港区 Apple ID。自己注册一个非中国区账号,不用绑定付款方式。实在搞不定就用免费的 sing-box 或 Loon。

订阅怎么自动更新?

点订阅那一行 → 打开 Auto Update,间隔设 1 小时 (和我们上游刷新频率一致)。

怎么挑最快的节点?

点任意服务器 → Latency Test (闪电图标)。或长按订阅 → Latency Test All 批量测。

连上了但打不开网页

VPN 开关关了再开一次。还是不行就换节点。VLESS+Reality 和 Hysteria2 在 iOS 上偶尔需要调分流规则。
