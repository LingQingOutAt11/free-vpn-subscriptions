🪜 操作步骤

  1. 安装 Clash Verge

    去 Clash Verge Rev 下载最新版本。Windows 选 .msi,macOS 选 .dmg,Linux 选 .deb/.rpm/.AppImage。双击安装后打开。

  2. 打开订阅面板

    点左侧栏的 订阅 (Profiles)。第一次打开是空的。点顶部的 + 或 导入。

  3. 粘贴订阅链接

    从首页复制 Clash 的订阅 URL,粘贴到 URL 输入框。名称随便填。点 保存,几秒后配置下载完成。

  4. 启用订阅

    点一下刚才的订阅卡片,让它变成激活状态(边框变蓝)。切到 代理 (Proxies) 标签,能看到选择器组里列出了所有免费节点。

  5. 打开系统代理

    回到主页,打开 系统代理 或 TUN 模式 开关。系统代理只对浏览器生效;TUN 模式接管所有流量(包括终端/游戏),首次启用需要管理员权限。

  6. 验证

    浏览器打开 ipinfo.io,显示的国家应该和你选的节点一致。如果还是国内 IP,检查系统代理开关,以及浏览器有没有绕过系统代理(Firefox 要手动勾 "使用系统代理设置")。

💡 常见问题排错

选哪个节点最好?

先用选择器组里的 URL-Test (自动选最快),它会按延迟挑最快那个。觉得慢就手动换一个国家。

怎么更新订阅?

右键订阅卡片 → 更新。或在订阅设置里开自动更新,建议间隔 1 小时(上游每小时刷新)。

只用浏览器不想开 TUN 可以吗?

可以。只打开 系统代理 就行,大部分浏览器都会走。Chrome/Edge 默认走系统代理;Firefox 要在网络设置里勾 "使用系统代理"。

提示 "profile update failed" 订阅更新失败

通常是 GitHub raw 临时抽风,等几分钟再试。如果一直失败,把订阅 URL 粘到浏览器看能不能打开,确认网络可达。
